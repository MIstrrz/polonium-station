Class GasSpawnPowerConsumerComponent

An entity with this will produce some amount of gas over time if supplied with power. Gas is output at a regular frequency, and the amount of gas spawned scales with the amount of power given. At high power input, gas returns diminish logarithmically. Expected to be used with a GasCanister that can contain the mixture.
